# 缓存区文本表示法
一些示例通过显示文本的修改前和修改后版本来描述对缓冲区内容的修改。这些示例在包含缓冲区名称的两条破折号之间显示所讨论的缓冲区的内容。此外，“∗”表示点的位置。（当然，点的符号不是缓冲区中文本的一部分；它表示点当前位于两个字符之间的位置。）  

	---------- Buffer: foo ----------
		This is the ∗contents of foo.
	---------- Buffer: foo ----------

	(insert "changed ")
		⇒ nil
	---------- Buffer: foo ----------
	This is the changed ∗contents of foo.
	---------- Buffer: foo ----------
