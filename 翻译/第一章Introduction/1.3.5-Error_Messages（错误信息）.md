# 错误信息
有些示例会触发错误。这通常会在回显区域显示一个错误消息。我们在以“error→”开头的行上显示错误消息。请注意，“error→”本身并不会出现在回显区域。  

	(+ 23 'x)
	error→ Wrong type argument: number-or-marker-p, x
